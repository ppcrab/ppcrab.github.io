Hexo博客的搭建其实比较简单，所以详细的步骤就不再赘述了，但是在这其中也遇到了一些坑，记录一下。并且这是我的第一个博客文章，感觉还不错。
我在搭建博客的时候主要是参考了这篇博客,这篇博客的步骤十分详细，基本上不需要多长时间就能搭建完成自己的hexo博客，但是在换主题的时候有一些坑一定要注意。

在Hexo官网上有一些主题不能用！一定要注意。

我用了好几个都是不能在本地显示CSS的更不用说是访问网页了，最后我还是找的比较成熟的主题next（搭建的方法很容易就能搜的到）并且我本人觉得这个主题还不错。
还有一个我们蓝鲸社团大佬的博客,用作参考也是一个十分不错的选择。

里面有点小问题


这里是有解决的方法的：
1.把主页配置文件_config.yml 里的post_asset_folder:这个选项设置为true
2.在你的hexo目录下执行这样一句话npm install hexo-asset-image –save，这是下载安装一个可以上传本地图片的插件，来自dalao：dalao的git
3 等待一小段时间后，再运行hexo n “xxxx”来生成md博文时，/source/_posts文件夹内除了xxxx.md文件还有一个同名的文件夹，这个文件夹的作用是来放置博客中引用到的图片文件的
另外，在使用canvas_nest的时候，要注意的一点是修改next主题配置文件的时候，不需要再手动添加
