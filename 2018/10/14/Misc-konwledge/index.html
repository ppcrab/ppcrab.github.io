outguess的使用方式：

加密：
outguess -k “my secret key” -d hidden.txt demo.jpg out.jpg
加密之后，demo.jpg会覆盖out.jpg,
hidden.txt中的内容是要隐藏的东西

解密：
outguess -k “my secret key” -r out.jpg hidden.txt
解密之后，解密内容放在hidden.txt中

xrange()

需要首先说明的是，这个函数在p2中使用，在py3中这个函数完全可以使用range（）进行替代，所以会报错

这个函数与range完全相同，不同的是，生成的不是一个数组，而是一个生成器。

参数说明;

  • start: 计数从 start 开始。默认是从 0 开始。例如 xrange(5) 等价于 xrange(0， 5)
  • stop: 计数到 stop 结束，但不包括 stop。例如：xrange(0， 5) 是 [0, 1, 2, 3, 4] 没有 5
  • step：步长，默认为1。例如：xrange(0， 5) 等价于 xrange(0, 5, 1)
